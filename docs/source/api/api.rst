
Interface de programmation d'application (API) de ``colibri``
=============================================================

mixins
------

Le sous-paquet (ou sous répertoire) ``mixins`` contient tous les modules
(composés de classes) dont la vocation est de permettre un héritage multiple
par les autres classes du paquet `colibri`.

utils
-----

Le sous-paquet (ou sous répertoire) ``utils`` contient tous les modules
(fonctions, classes, etc.) dont la vocation est d'aider au bon fonctionnement du
paquet `colibri` sans être des pièces centrales.
